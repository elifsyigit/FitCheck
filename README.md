
**FitCheck** is a browser extension powered by a Python backend using Firebase Cloud Functions. This project integrates a frontend extension with serverless architecture to process user requests.

## 🚀 Features

- **Browser Extension:** Custom frontend interface for user interaction.
- **Serverless Backend:** Python-based logic hosted on Firebase Cloud Functions.
- **Real-time Database/Storage:** Uses Firebase for data management.

## 🛠️ Tech Stack

- **Frontend:** JavaScript, HTML, CSS (Chrome Extension)
- **Backend:** Python (Google Cloud Functions / Firebase)
- **Infrastructure:** Firebase (Hosting, Firestore, Functions)

## 📂 Project Structure

```text
FitCheck/
├── extension/       # Browser extension source code (Manifest, JS, HTML)
├── functions/       # Backend logic (Python Cloud Functions)
├── firebase.json    # Firebase project configuration
└── README.md        # Project documentation
```
